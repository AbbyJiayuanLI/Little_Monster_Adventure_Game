# mario
Mario Game with C++/Qt

# Demo
Check [this](https://www.youtube.com/watch?v=qZN8QkhRMWE) out on YouTube! 


## How to run the game?
1. Open adventure.pro file in QT creator.
2. Run the main.cpp file. There should be a pop-up game window after this.
3. Enjoy game!
